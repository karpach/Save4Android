In order to install a JavaScript in the Scripts menu, place it in the Scripts folder (Photoshop CS5 /Presets /Scripts).

SaveIcon4Android.jsx - Generates icons for android in respective folders:

480dpi 144x144 - drawable-xxhdpi
320dpi 96x96 - drawable-xhdpi
240dpi 72x72 - drawable-hdpi
160dpi 48x48 - drawable-mdpi
120dpi 36x36 - drawable-ldpi

SaveImage4Android.jsx - Generates images for android in respective folders, where dimensions are proportional to selected resolution / dpi.

For example:

Resolution: 480dpi
Width:150
Height:80

Becomes:

480dpi 150x80 - drawable-xxhdpi
320dpi 100x53 - drawable-xhdpi
240dpi 75x40 - drawable-hdpi
160dpi 50x27 - drawable-mdpi
120dpi 37x20- drawable-ldpi
